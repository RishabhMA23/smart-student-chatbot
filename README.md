# smart-student-chatbot
An AI-powered chatbot designed to assist students with academic queries, basic guidance, and conversational support using Natural Language Processing.
## Features
- Responds to student-related queries
- Uses simple NLP pattern matching
- Easy to expand with more intents

## Tech Used
- Python
- JSON

## How to Run

1. Clone the repo:
git clone https://github.com/your-username/smart-student-chatbot.git

2. Navigate:
cd smart-student-chatbot

3. Run:
python app.py
